Evaluare Nr. 2 - Git Repository Management

I. Commit Etape

1. Clonare Repository

git clone https://github.com/MacreiAndrei/evaluarenr2.git

2. Adăugare fișier settings.xml

git add settings.xml git commit -m "Commit file settings.xml"

3. Redenumire settings.xml la configuration.xml

git mv settings.xml configuration.xml

II. Branch Management

1. Creare Branch Stage

git checkout -b stage

2. Creare Branch Dev

git checkout -b dev

3. Redenumire Branch Dev la Develop

git branch -m dev develop

Concluzie

Toate etapele cerute au fost îndeplinite cu succes, incluzând:

  • Crearea și gestionarea repository-ului
  • Adăugarea și modificarea fișierelor
  • Lucrul cu branch-uri diferite
  • Actualizarea repository-ului local și remote
